电话整蛊

五月 05, 2019

今天周日，不过我得上班，她得上课，因为是五一节才放假回来，所以周日也得补之前放掉的假，算一下上周补了一天，今天又补一天，那五一不就相当于放了两天周末？只不过是将它们连在一起放的而已，所以才会有四天假。
今天的她坏坏的，从早上起来就开始在酝酿着来整我，因为这个小傻瓜感冒了，声音变化太大，所以就想打电话来整我。我下班之后接到她的电话时，还真没听出来是她，她在那边都笑出了声(我没听到她笑出了声)……地铁上确实有点吵，要在安静的环境下我早就认出来了她的声音了，哼😒，不过这样的整蛊真的挺好玩的，哈哈。
之前她给了我她买的饼干，我今早看的时候我以为还过期了呢，我就跟小可爱说了，她便去找了客服，结果是我给弄错了，尴尬😂，我打算之后上班当早饭吃，嘻嘻。
她早上的时候光速打自己的脸😂，事情是这样的，我早上跟她说我打瞌睡了，她说上课的时候她没打瞌睡，结果没一会第二节的时候一下就打瞌睡了，脑袋一点一点的，哈哈哈哈，好可爱。
晚上她跟她的室友一起出去逛了下， 看了个电影，回来还在跟我说小男主角挺帅的，哈哈，小小花痴。


摄影于：2019-06-02 地点：乐和乐都
