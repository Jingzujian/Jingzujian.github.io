相约缙云山

十一月 03, 2019

我一直很想跟她去爬山，毕竟我俩都缺少运动，这一次我们终于相约了北碚缙云山。
我头天因为发小过生，所以睡得比较晚，但也不影响我第二天起来就去找她的冲动，我很想跟她一起在学校里吃早饭，毕竟这样的机会可不多呐。
我大概九点多钟就到了她们学校，跟她一起吃了早饭，然后坐车去了缙云山，大概十点半左右到达了目的地，我们沿着上山的道路走着走着，后来我发现一条小道，我们原本打算从那边下山，结果走到了农家院，到处都是够，吓得小可爱都不敢继续往前走了，我一直给她做工作让她不要怕不要怕，其实我心里还是有点怕的(毕竟有一条黑狗，很大一只那种)，后来我们只能沿着原路返回了😂
我跟她约好了以后每个月至少爬一次山，但是似乎到现在也只爬了一次还是两次…
